# Secure-Bank-Management-System
A browser-based Bank Management System built with HTML, CSS, and JavaScript featuring user authentication, balance management, deposits, withdrawals, and transaction history using Local Storage.


## Features

- User Registration
- Secure Login
- Deposit Money
- Withdraw Money
- Balance Tracking
- Transaction History
- Local Storage Authentication

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage

## Project Structure

```
index.html
register.html
dashboard.html
style.css
script.js
```
